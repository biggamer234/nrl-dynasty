# NRL Dynasty

Playable NRL-themed dynasty game with Supabase-ready auth, cloud-save scaffolding, club history, best-ever teams, and State of Origin mode.

## Play
Open `nrl-dynasty_publish_ready.html` in a browser or publish `index.html` through GitHub Pages.

## Setup
1. Replace `YOUR_SUPABASE_URL` and `YOUR_SUPABASE_ANON_KEY` in the HTML.
2. Create your Supabase tables for saves, club history, and Origin history.
3. Deploy the HTML file as `index.html` at the repo root or in `/docs`.
